🔬 Capabilities

What Makes It Different

Every feature derives real signals from your document — not from templates or hardcoded examples.

🔬

Adaptive OCR Pipeline

OpenCV preprocessing selects between adaptive Gaussian thresholding and Otsu binarization automatically based on which produces cleaner output. Per-word confidence filtering (≥60%) discards OCR noise before NLP runs.

OpenCV Tesseract pdf2image 300 DPI
🏢

NLP Entity Extraction

spaCy ORG and DATE entities identify real companies and date spans. Noun-chunk analysis detects 2–6 word job title phrases. A 40+ word noise filter removes section headers, phone numbers, and numeric strings.

spaCy NER en_core_web_sm
📈

Deterministic Scoring

Five independent scorers produce component scores: Experience (30%), Skills (25%), Content Quality (20%), ATS Compatibility (15%), Education (10%). Weighted composite gives an 0–100 score with A+–F grade.

No randomness Same input = same score
🤖

Google Gemini AI

On-demand layer using gemini-2.0-flash. Returns 10 structured fields: summary, career level, strengths, weaknesses, improvements, ATS tips, project suggestions, recommended roles, red flags, unique insights.

gemini-2.0-flash SHA-256 cache graceful degradation
🗄️

Structured MongoDB Storage

Every analysis saved with nested schema: meta, score, features (skills/experience/content/ats), ai_insights, insights. Raw file stored in GridFS with original filename. Three indexes for fast history queries.

Motor async GridFS 3 indexes
📊

Interactive Dashboard

ECharts score gauge, radar chart for 7 section scores, donut chart for skill category distribution, horizontal bars for keyword frequency. Dark/light mode, real-time AI suggestion button, download original file.

ECharts v6 React 18 Dark mode
